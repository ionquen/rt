Что такое регулярные выражения

Определение

Регулярные выражения - специальные шаблоны, используемые для нахождения в строке необходимых подстрок. Сам шаблон представляет из себя строку, состоящую из специальных символов, каждый из которых имеет определённое значение. Такие символы называются метасимволами. В этой статье рассматриваются метасимволы, флаги и множества символов, используемые в регулярных выражениях, которые могут пригодиться для создания регулярных выражений.

Специальные символы

Шаблон регулярного выражения строится из любых символов, некоторые из которых обладают специфическим назначением. Все цифры и символы алфавита, с которыми работают спецсимволы, записываются как есть. Для того, чтобы записать метасимвол как есть, его необходимо экранировать обратной косой чертой (обратный слеш).

Квантификатор - это метасимвол, позволяющий ограничить число повторений предшествующего символа или группы символов. Существуют следующие квантификаторы: ? + * {n}.

СимволПример шаблонаОписание
//.*/Создание регулярного выражения. Символ должен находиться в начале и в конце шаблона.
./.*/Представляет из себя любой символ в строке.
*/.*/Квантификатор. Находит строку, состоящую из предыдущего символа (в данном случае любой символ), который повторяется 0 или более раз. В текущем примере равнозначен всей строке, по которой осуществляется поиск
{ }/a{2}/Квантификатор. Находит строку, состоящую из предыдущего символа, который повторяется указанное в фигурных скобках количество раз. Можно задать диапазон. Так диапазон {2, 5} найдёт строки длиной от 2 до 5 символов, а {2, } найдёт строку, длина которой больше или равна 2.
+/.+/Квантификатор. Находит строку, состоящую из предыдущего символа, который повторяется 1 или более раз.
?/.?/Квантификатор. Находит строку, состоящую из предыдущего символа, который повторяется 0 или 1 раз.
?/.*?/Изначально квантификаторы . и + жадные, то есть стараются захватить в результирующую строку как можно больше символов. В данном случае квантификатор, после которого стоит метасимвол ? становится нежадным и будет искать самые короткие строки соответствующие шаблону, например в строке "abcde" найдёт только a
^/^a+/Указывает на то, что последующая часть шаблона должна сопоставиться с началом строки. В данном шаблоне при сопостовлении со строкой "aabc" найдёт aa, в строке "ddaaa" ничего не найдёт.
$/a+$/Указывает, что предшествующая часть шаблона должна сопоставиться только в конце строки. Здесь в строке "cba" найдёт a
[ ]/[abc]*/Набор символов, указанный в квадратных скобках. Можно указать диапазон через тире, например /[a-z]*/, которые равен любому слову, состоящему из букв латинского алфавита. Также имеется возможность вставлять символьные классы: /[\d.]*/ что соответствует любому числу в строке. Чтобы добавить символ тире в набор необходимо его разместить в конце набора /[a-z-]*/.
[^ ]/[abc]*/Символы, которые не входят в данный набор символов. Символ ^ добавляется в группу только если он находится не в начале набора.
()/(abc)+/Скобочная группа. Группирует содержимое и позволяет применить к ним квантификаторы. Содержимое скобок можно получить в результате применения шаблона.
(?:)/(?:abc)+/Исключающая скобочная группа. Содержимое скобок сключается из результата.
x(?=y)/a(?=b)/Сопоставляется с x, только если за x следует y
x(?!y)/a(?!b)/Сопоставляется с x, только если за x не следует y
x(?<=y)/a(?<=b)/Сопоставляется с x, только если за y следует x
x(?<!y)/a(?<!b)/Сопоставляется с x, только если за y не следует x
a|b/a(a|b|c)a/Альтернация. Позволяет сопоставиться либо с a, либо с b. В данном примере найдёт aaa и aba в строке baaabac.
(?<key>)/(?<ссылка>[a-zA-Z]{2})(\k<ссылка>*)/Создаёт ссылку на скобочную группу, позволяя получить её значение в результате по ключевому слову.
\/\\.+/Символ экранирования - позволяет использовать спецсимволы в шаблонах. При применении на буквы латинского алфавита представляет собой символьные классы (об этом ниже). В данном примере находит строку, начало которой равно точке.

Флаги

Флаги указывают правила сопоставления шаблона и указываются после регулярного выражения: /abc/i. Также можно установить несколько флагов неважно с какой последовательностью.

ФлагРасшифровкаЗначение
uUnicodeОбработать строку по кодировке Unicode
gGlobalИскать все совпадения. При отсутствии флага находит только первое попавшееся совпадение
iIgnoreИгнорировать регистр при сопоставлении
mMultilineПроизводит поиск по нескольким строкам. Шаблон, в котором используется $ и ^ применяется в том числе и для каждой строке

Символьные классы и ссылки

Для указания группы символов существует спецсимволы [], однако для некоторых групп изначально предопределены символьные классы, ускоряющих создание шаблонов.

Для вставки символьного класса используется знак обратной косой черты \.

Пример: /\d/ - найти все целые числа.

СимволРасшифровкаЗначение
\dDigitЦифры от 0 до 9.
\sSpaceЛюбые пробельные символы, включая символ пробел, табуляция, перевод строки.
\wWordЛюбая буква латинского алфавита, а также цифры от 0 до 9 и символ подчёркивания.
\tTabulationСимвол табуляции.
\rCarriage returnВозврат каретки - символ, управляющий возвращением текущей позиции в начало текущей линии.
\nNew lineСимвол переноса строки.
\rForm feedРазделитель страниц.
\vVertical tabulationСимвол вертикальной табуляции.
\bBorderГраница слова.
[\b]BackspaceСимвол Backspace. Задаётся только в символьном наборе.
\xCCПредставляет из себя символ с кодом CC.
\uCCCCUnicodeПредставляет из себя Unicode-символ с кодом CCCC.
\p{prop}UnicodeПредставляет из себя символ, свойство prop которого соответствует определённой группе Unicode-символов.
\k<>KeywordОбратная ссылка по названию якоря. Пример: /(?<word>[a-zA-Z]{2})(\k<word>*)/ - создаётся ссылка на скобочную группу, значение которой повторно применяется впоследствии.
\nОбратная ссылка на скобочную группу, где n - число от 1, представляющее номер скобочной группы в последовательности.
\0NullСоответствует символу Null

Можно также находить символы, не состоящие в данном символьном классе. Для этого символ записывается в верхнем регистре.

Например: /\D/ - найти все символы, не состоящие в классе d, то есть не цифры.

Свойства символов

Не многие знают, но каждый Unicode символ обладает некоторым свойстом, что позволяет отнести его к определённой группе символов: цифры, числа, алфавит, эмодзи, наличие символа в кодировке ASCII и так далее.

Для поиска по свойству применяется следующий синтаксис: /\p{prop=value}/, где prop представляет собой свойство, а value его значение, например: {sc=Cyrillic}. Если значение свойства является булевым, то достаточно указать только свойство, например: {Alpha}.

Для того, чтобы применить данный функционал, необходимо к регулярному выражению добавить флаг u.
