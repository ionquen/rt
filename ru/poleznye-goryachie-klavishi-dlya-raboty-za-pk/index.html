Полезные горячие клавиши для работы за ПК

~4 мин.

Горячие клавиши являются спасением для большинства пользователей ПК, ведь именно благодаря им экономится значительная часть времени при работе за компьютером. В этой статье вы узнаете о самых полезных горячих клавишах, способных повысить вашу скорость работы за компьютером на более высокий уровень.

Обязательно посмотрите на наш приготовленный список, даже если вы уверены, что знаете хотя бы половину из имеющихся здесь комбинаций. Мало ли, есть то, что вы могли бы использовать в своей повседневности. И знайте, что использование горячих клавиш гарантировано сократит время, которое вы сейчас может используете напрасно, например любое действие, совершаемое с использованием мыши.

Горячие клавиши для Windows

В основном это горячие клавиши для работы с текстом, однако существуют различные комбинации клавиш, которые являются полной заменой аналогичным действиям, выполняемые с помощью мыши.

Советуем научиться их использовать и тогда работа с текстом станет для вас невероятно лёгкой и приятной.

Работа с текстом

Горячие клавишиОписание
Ctrl + CКопирование выделения в буфер обмена (текст, изображения и так далее)
Ctrl + XВырезать выделение в буфер обмена
Ctrl + VВставка содержимого из буфера обмена
Ctrl + AВыделить всё (весь текст, всю страницу, выбрать все файлы в папке ...)
Ctrl + ZОтмена действия
Ctrl + YПовторить отменённое действие
Ctrl + ShiftПереключить раскладку клавиатуры
HomeПереместить курсор в начало строки (или выделить первый файл в каталоге)
EndПереместить курсор в конец строки (или выделить последний файл в каталоге)
Ctrl + ←→Переместить курсор в начало/конец следующего/предыдущего слова
Ctrl + Shift + ←→Переместить курсор в начало/конец следующего/предыдущего слова и добавить его в выделение. Помогает очень быстро выделят текст
Shift + ←→Переместить курсор на предыдущую/следующую строку и добавить это в выделение
Shift + ↑↓Переместить курсор на предыдущий/следующий символ и добавить его в выделение
Windows + VОткрыть историю буфера обмена (содержит всю недавно добавленную в буфер информацию, позволяет переключаться между собой)

Взаимодействие с каталогами

Горячие клавишиОписание
Ctrl + D (или Delete)Удалить файл в папке
Ctrl + R (или F5)Обновить содержимое окна (обновить вкладку)
Ctrl + Shift + NСоздать новую папку
BackspaceПоднятся на каталог выше
Ctrl + NОткрыть текущий каталог в новом окне (будет открыто несколько одиннаковых окон)
F3Поиск в каталоге (или на сайте)
Alt + ←→Действие назад/вперёд. Предыдущая/Следующая в истории страница сайта или каталог
Alt + EnterОтобразить окно свойств выбранного файла

Открытие системных окон

Горячие клавишиОписание
Ctrl + Shift + EscОткрыть диспетчер задач
Ctrl + EscОткрыть меню Пуск
F2Переименовать выделенный файл (или элемент)
F11Развернуть на весь экран/Свернуть в небольшое окно
Windows + DОтобразить/Скрыть рабочий стол
Windows + EОткрыть папку Проводник
Windows + LБлокрировка (смена) учётной записи
Windows + IОткрыть меню Параметры
Windows + RОткрыть окно Выполнить
Windows + SОткрыть окно Поиск
Windows + Pause/BreakОткрыть свойства Системы

Взаимодействие с окнами

Горячие клавишиОписание
Windows + MСвернуть все окна
Windows + Shift + MВосстановить свёрнутые окна
Windows + ↑↓Развернуть/Свернуть окно
Windows + ←→Переместить окно в левую/правую часть экрана
Windows + ←→ + ↑↓Переместить окно в левый/правый верхний/нижний угол экрана
Shift + Нажатие кнопки панели задачОткрыть другой экземпляр приложения в новом окне
Alt + TABПереключение между активными окнами
Alt + F4 (Ctrl + W)Закрыть текущее окно
Alt + SpaceОткрыть контекстное меню текущего окна

Создание скриншотов

Горячие клавишиОписание
PrtScnСоздать снимок и добавить его в буфер обмена
Alt + PrtScnСоздать снимок только текущего окна и добавить его в буфер обмена
Windows + Shift + SСоздать снимок части экрана (можно будет выбрать область)
