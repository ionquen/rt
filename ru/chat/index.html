О программе

Онлайн чат с передачей файлов

~2 мин.

Данный сервис предлагает пользователям обмениваться сообщениями, включая файлы, между собой, напрямую, без каких-либо посредников в виде социальных сетей и мессенджеров. Всё это происходит прямо в браузере.

Почему стоит попробовать?

  1. Никакой регистрации и авторизации.
  2. Поддержка передачи неограниченного числа файлов.
  3. Никаких ограничений в 50 гигабайт и прочее, что есть у каждого другого обменника.
  4. Безопасность. Вы общаетесь в защищённом канале напрямую, то есть из браузера на одном компьютере в браузер на другом.
  5. Простота и удобство. Вам нужно просто поделиться ссылкой или собеседник должен просканировать ваш QR код и будет добавлен в список подключений.
  6. Не надо скачивать никаких посторонних программ. Всё работает в режиме онлайн через браузер устройства. Через наш сервис вы можете общаться со знакомыми на любую тему и обмениваться любыми файлами. Из-за прямого соединения между вами содержимое переписки останется только между вами. Оно не сохранится нигде, впрочем если сами участники соединения не сохранят информацию к себе.

Как это работает?

Наш сервис работает с использованием технологии webRTC. Подключиться напрямую невозможно из-за исторических особенностей Интернета, для начала браузеры должны передать друг другу информацию о себе. Это конфигурационные данные, и их кто-то должен доставить друг другу, ведь соединение всё еще не установлено.

В этом случае на помощь приходит сторонний сервер, через который будут направлены между контактами данные для налаживания соединения. Мы предоставляем для этого свой сервер. Пользователи подключаются к нему, обмениваются данными для организации соединения, и как только соединение напрямую будет установлено отключаются от нашего сервера. Этот сервер сводит пользователей друг с другом.

Как передать большой файл?

Наш онлайн чат предоставляет возможность передачи любого типа файлов по созданному каналу между пользователями. Это также означает, что вы можете передать файл любого размера друг другу, без ограничений. Стоит помнить, что если соединение будет прервано, то придётся отправлять файл заново.

Предостережение

Важно отметить следующее перед тем, как вы начнёте использовать этот сервис: не рекомендуется устанавливать контакт с людьми, которые не вызывают доверия. Так как соединение устанавливается напрямую между браузерами, контакт может узнать ваш ip адрес, а также другие сведения, отправляемые друг другу для установки соединения, например информация связанная с браузерами и так далее.
