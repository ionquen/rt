Online Calculator - Convenient and accurate

Calculator online

This online calculator allows you to perform any calculations, ranging from simple arithmetic operations to complex expressions involving trigonometry. It can be useful everywhere: at home, at work or at school.

It is suitable for any activity: for engineering calculations, for solving problems by formulas, for quickly finding the sum and difference, product and quotient of any numbers.

There is support for such actions as: exponentiation, finding the root, finding the percentage of the number and using parentheses to indicate the sequence of calculations. We have provided a lot of nice features for users:

  1. User-friendly, minimalistic, with large buttons interface
  2. History of calculations
  3. Memory cells
  4. Can work without internet
  5. A lot of little things that contribute to the pleasant use of the calculator.

Our calculator is adapted for different devices, such as tablets, smartphones, personal computers.

On our website there is support for full-screen mode for the calculator.

Memory Cells

This calculator supports a convenient extended system of memory cells. During calculations, you will be able to save intermediate results in memory cells through a special button or hotkey, and then apply the saved value in other calculations.

This functionality is useful when you need to temporarily store intermediate values somewhere. Previously, the only memory cell available in the calculator was used for this, or the result was copied into a notepad or other text document. Here it is enough to add this value to a special list. This list can contain up to 10 values.

To insert a saved number, click on this number in the sidebar where these values are saved.

Computing history

The online calculator also provides an automatically saved calculation history.

Memory cells and history are available after the site page is reloaded and they can always be cleared by clicking on the appropriate buttons.

Historical facts

The calculator as a computing device has a history of more than 300 years. Throughout history , such devices have been created as:

  1. Adding machines that find the sum of numbers between the values entered and stored in the device.
  2. Tabulators that process information recorded on punched cards.
  3. Arithmometers designed to accurately perform simple arithmetic operations: addition, subtraction, multiplication, division.
  4. Electronic calculators powered by electricity.
