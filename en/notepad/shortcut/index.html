Useful keyboard shortcuts for working at a PC

Hotkeys are a salvation for most PC users, because it is thanks to them that a significant part of the time is saved when working at the computer. In this article, you will learn about the most useful keyboard shortcuts that can increase your computer speed to a higher level.

Be sure to look at our prepared list, even if you are sure that you know at least half of the combinations available here. You never know, there is something that you could use in your daily life. And be aware that using hotkeys is guaranteed to reduce the time that you can now use in vain, for example, any action performed using the mouse.

Keyboard shortcuts for Windows

Basically, these are keyboard shortcuts for working with text, but there are various keyboard shortcuts that are a complete replacement for similar actions performed with the mouse.

We advise you to learn how to use them and then working with the text will become incredibly easy and pleasant for you.

Working with text

Keyboard shortcutsDescription
Ctrl + CCopy selection to clipboard (text, images, etc.)
Ctrl + XCut selection to Clipboard
Ctrl + VPasting content from the clipboard
Ctrl + ASelect everything (all text, the whole page, select all files in the folder...)
Ctrl + ZCanceling an action
Ctrl + YRepeat the canceled action
Ctrl + ShiftSwitch keyboard layout
HomeMove the cursor to the beginning of the line (or select the first file in the directory)
EndMove the cursor to the end of the line (or highlight the last file in the directory)
Ctrl + ←→Move the cursor to the beginning/end of the next/previous word
Ctrl + Shift + ←→Move the cursor to the beginning/end of the next/previous word and add it to the selection. Helps to highlight text very quickly
Shift + ←→Move the cursor to the previous/next line and add this to the selection
Shift + ↑↓Move the cursor to the previous/next character and add it to the selection
Windows + VOpen the clipboard history (contains all the information recently added to the clipboard, allows you to switch between each other)

Interaction with directories

Keyboard shortcutsDescription
Ctrl + D (or Delete)Delete a file in a folder
Ctrl + R (or F5)Refresh window contents (refresh tab)
Ctrl + Shift + NCreate a new folder
BackspaceThey will go up to the catalog above
Ctrl + NOpen the current directory in a new window (several dialog boxes will be opened)
F3Search in the catalog (or on the website)
Alt + ←→Action back/forward. Previous/The next page in the history of the site or catalog
Alt + EnterDisplay the properties window of the selected file

Opening system windows

Ctrl + EscOpen the Start menu

Windows + DDisplay/Hide desktop
Windows + EOpen the Explorer folder
Windows + LAccount adjustment (change)

Windows + ROpen the Run window

Interaction with windows

Keyboard shortcutsDescription
Windows + MMinimize all windows
Windows + Shift + MRestore minimized Windows
Windows + ↑↓' Expand/Minimize Window
Windows + ←→Move the window to the left/right side of the screen
Windows + ← → + ↑↓Move the window to the left/right upper/lower corner of the screen
Shift + Pressing the taskbar buttonOpen another instance of the application in a new window
Alt + TABSwitching between active windows
Alt + F4 (Ctrl +W)Close the current window
Alt + SpaceOpen the context menu of the current window
